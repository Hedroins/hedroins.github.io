代码嵌入网页的方法

网页中嵌入 JavaScript 代码，主要有三种方法。

  • <script>元素直接嵌入代码。
  • <script>标签加载外部脚本
  • 事件属性
  • URL 协议

script 元素嵌入代码

就是把javascript脚本直接写在script标签中，type属性可以指定脚本类型。通常可以设置为text/javascript（用于较老的浏览器）和application/javascript（用于较新的浏览器）
